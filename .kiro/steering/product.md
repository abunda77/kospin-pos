# Product Overview

This is a **Point of Sale (POS) Kasir System** built for retail businesses in Indonesia. The application provides comprehensive retail management capabilities including:

## Core Features
- **Product Management**: Inventory tracking, categories, barcode generation
- **Sales Processing**: POS interface, order management, receipt printing
- **Payment Integration**: Multiple payment gateways (Midtrans, Xendit)
- **Member Management**: Customer loyalty and membership system
- **Financial Tracking**: Expense management, sales reporting, data trending
- **Admin Panel**: Filament-based administrative interface
- **Thermal Printing**: ESC/POS thermal printer support (58mm)
- **Data Management**: Excel import/export, backup system
- **API Documentation**: Auto-generated with Scramble

## Target Users
- Small to medium retail businesses
- Indonesian market (documentation in Bahasa Indonesia)
- Businesses requiring thermal receipt printing
- Multi-user environments with role-based permissions
